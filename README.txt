Drush archiver
---------------
Drush archiver is an extension to drush to handle archives.
To most of the code bases on the archiver api of drupal.
See http://api.drupal.org/api/search/7/archiver.
The archiver api itself includes a lot code of Vincent Blavet's <vincent@phpconcept.net>
PhpConcept Zip/Tar Libraries.
